Templates for Journal Entries

User can configure journal entries templates, useful for recurring entries. The amount of each template line can be computed (through python code) or kept as user input. If user input, when using the template, user has to fill the amount of every input lines. The journal entry form allows lo load, through a wizard, the template to use and the amounts to fill.
