Project Cost Analysis

Eficent

== Analyze the costs of your project == Eficent brings you this module to better analyze the costs of your projects.

The aim is to enable the project manager can see at all times the relationship between the planned cost, budget, current theoretical cost actual cost of the project.

The following reports are defined:

Resource usage analysis: For each analytical account associated (taking into account the child accounts) breaks down the following information: Analytical Account Code, Analytical Account Name, Planned Cost, Budgeted cost, Commited cost, Current Actual Cost Resource usage (amounts): Provides a chart view of the resource usage per analytic account, based on amounts. Resource usage (quantities): Provides a chart view of the resource usage per analytic account, based on quantities.

Resource usage analysis by product: For each analytical account and product associated (taking into account the child accounts) breaks down the following information: Analytical Account Code, Analytical Account Name, Planned Cost, Budgeted cost, Commited cost, Current Actual Cost Resource usage by product (amounts): Provides a chart view of the resource usage per analytic account and product, based on amounts. Resource usage by product (quantities): Provides a chart view of the resource usage per analytic account and product, based on quantities.

The following PDF forms are defined:

Analytic Planning Balance: Provides the total credit, debit, balance and quantiy for the planned, budgeted, commited and actual costs and revenues.
