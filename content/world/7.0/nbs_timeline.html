So what does this module do?

This is the base module, by itself it doesn't do much. But you can use it as the basis for your own modules.

There are two example modules (for now) which you can intstall:

  • nbs_timeline_lead
  • nbs_timeline_partner

How can I use it in my own modules?

Easiest way is to look at the example modules. We promise to make some documentation in the near future!
