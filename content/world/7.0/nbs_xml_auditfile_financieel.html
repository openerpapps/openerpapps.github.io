Neobis XML Auditfile Financieel

This module exports financial (accounting) information from your OpenERP system to a .XAF file.

This module is considered to be in Beta

It does so according to the standard set by the dutch tax authorities.

This module is tested with the default OpenERP dutch accounts (l10n_nl). But it should work with any config. It implements the minimum set needed to generate a valid XML. Most optional information will be added in future releases.

This modules uses the xml_template module. created by Swing Entwicklung betrieblicher Informationssysteme GmbH, so thanks for that!

You can export a audit-file by going to:
  • Accounting-->Reporting-->Legal reports->Accounting Reports->Export XML Auditfile

Feel free to test and contribute to this module!
