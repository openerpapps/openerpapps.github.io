Welcome to OpenERP Apps.

We provide a `pip` iinstallable index for official and community OpenERP apps and modules.

Example to install a `myopenerp` environment with the CRM app:


$ virtualenv --system-site-packages myopenerp
$ source myopenerp/bin/activate
$ pip install --extra-index-url http://openerpapps.info/simple/openerp/7.0 openerp-crm
$ openerp-server --addons-path=lib/python2.7/site-packages/openerp/addons
