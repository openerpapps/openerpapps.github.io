Quickstart

Installing OpenERP with the CRM app.

Create a virtualenv and install the CRM app package. Using --system-site-packages is not required, but will speed things up by reusing Python modules already installed in your system.

$ virtualenv --system-site-packages mydemo
$ cd mydemo
$ source bin/activate
$ pip install --extra-index-url http://openerpapps.info/simple/openerp/7.0 openerp-crm

Now fire up the instance installation and give it a test drive:

$ createdb mydemo
$ openerp-server -d mydemo -i crm --addons-path=lib/python2.7/site-packages/openerp/addons
$ firefox http://localhost:8069
